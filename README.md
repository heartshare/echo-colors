echo-colors
==========
echo with colors. Available are red, green, brown, blue, purple, cyan, gray.

Installation
----------

    $ cat echo-colors >> ~/.bashrc
    $ source ~/.bashrc  
	
Usage
----------

    $ echo-colors NG red  
    $ echo-colors OK green
